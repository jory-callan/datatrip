## 设计原则

- 按数据源类型分入口（SQL / NoSQL / Search / MQ），共享中间件链
- Pipeline 作为请求上下文贯穿始终，不可变字段与可变字段分离
- 中间件链永不中断，错误全量收集
- 审计日志作为中间件自动执行，不散落
